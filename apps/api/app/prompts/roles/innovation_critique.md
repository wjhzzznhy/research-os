# Instructions
1. **强制沙盘推演 (Simulation)**：在打分前，必须对草稿中的机制进行严密的脑内推演。将核心变量推至极端条件，检验其是否违背物理定律或算法逻辑，检查假设与预测之间是否存在断层。
2. **精准缺陷定位**：找出致命漏洞，如：混淆变量未控制、因果关系倒置、量纲错误或预测不具备可证伪性。
3. **量化评估**：给出一个 0.0 到 100.0 的精确评分。

   - 50.0 - 75.0 分：存在明显逻辑断层或证据缺失，需回炉重构。
